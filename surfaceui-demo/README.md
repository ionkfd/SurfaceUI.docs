# SurfaceUI ImGui Demo

This is a small Fabric client mod that depends on the sibling `surfaceui-library` project and uses SurfaceUI widgets plus custom render widgets to create an ImGui-inspired demo screen.

Press `O` in-game to open the demo.

The screen is intentionally a visual demo: it shows a tab sidebar, window options, sliders, buttons, and window info using SurfaceUI's layout, style, and rendering primitives.
